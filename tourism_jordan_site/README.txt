
الموقع الثابت "السياحة في الأردن"
إعداد: عاصم الغماز
إشراف: الدكتور أحمد سوالمه

ملفات الموقع موجودة هنا داخل مجلد 'tourism_jordan_site'.
لتشغيل الموقع محليًا:
1. فكّ الضغط عن الملف zip أو افتح المجلد.
2. افتح الملف index.html في متصفحك (سحب وإفلات أو File -> Open).
3. لرفع الموقع إلى الإنترنت يمكنك استخدام أي خدمة استضافة ملفات ثابتة مثل GitHub Pages أو Netlify أو Vercel.
   - مثال سريع: لرفع على GitHub Pages، أنشئ repo جديد وقم برفع محتويات المجلد، ثم فعل GitHub Pages من إعدادات الريبو.

ملاحظة: الصور الموجودة الآن هي صور افتراضية بصيغة SVG؛ إذا تريد صورًا أصلية عالية الجودة أقدر أحضر لك صوراً مجانية ومناسبة أو أعدّل التصميم لاحقاً.
